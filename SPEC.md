# 【AI Team Intro】規格計劃書

## 1. 專案概述

### 1.1 專案背景與目的

AI Agent 團隊日益龐大，但缺乏一個集中展示的窗口——內部成員不清楚彼此的能力邊界，外部合作夥伴無法快速了解 AI 能做什麼，投資人更難評估技術實力。本工具旨在建立一個動態的線上展示頁面，不只是靜態的名片牆，而是能即時反映 Agent 狀態、支援一鍵對話、並以視覺化方式呈現團隊協作關係的互動平台。透過這個入口，期望將 AI 能力的透明度提升至與傳統網站相同的水平，並成為對外溝通的核心樞紐。
### 1.2 目標受眾（TA）

- 內部團隊成員：不清楚其他 Agent 的能力範圍，跨 Agent 協作時容易重複勞動或溝通斷層
- 外部合作夥伴：需要快速評估 AI 能力是否符合專案需求，但沒有統一的入口獲取資訊
- 投資人：想展示技術實力，但缺乏具體、可互動的 Demo 入口
- 潛在客戶：希望在接觸前先了解團隊背景，增加信任感
### 1.3 專案範圍

### In Scope（做）

- Agent 卡片系統：姓名、角色、照片、能力標籤、狀態指示、聯絡頻道
- 團隊架構圖：組織圖呈現 Agent 協作關係，支援點擊查看詳情
- 即時對話入口：點擊 Agent 即可在對應頻道傳訊息
- 動態狀態廣播：WebSocket 即時更新在線/忙碌/離線狀態
- 使用案例展示：成功案例量化成果
### Out of Scope（不做）

- Agent 實際對話功能（僅提供入口，不內建聊天）
- Agent 能力即時測試沙盒
- 外部用戶註冊與權限管理
- 與其他內部系統（PMS、CRM）的深度整合
